# Projeto de Inteligência Artificial

## Predição do Nível de Liquidez em Fundos de Investimento

Este projeto tem como objetivo aplicar técnicas de Machine Learning para classificar o nível de liquidez de fundos de investimento com base em dados da ANBIMA.

## Tecnologias utilizadas
- Python
- Pandas
- Scikit-learn
- Matplotlib

## Estrutura do projeto
- relatorio/: relatório do projeto
- dataset/: base de dados utilizada
- codigo/: scripts e notebooks
- resultados/: outputs do modelo

## Modelo
Foi utilizado o algoritmo Random Forest para classificação do nível de liquidez dos fundos.
